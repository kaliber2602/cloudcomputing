Lộ trình học Web Server Nginx từ 0 đến 1

Nginx không chỉ là một web server, nó là một Reverse Proxy mạnh mẽ mà mọi Dev/DevOps cần phải biết.

Giai đoạn 1: Static Hosting

Bắt đầu bằng việc phục vụ các file HTML/CSS đơn giản như blog này. Hiểu về root và alias.

location /media/ {
    alias /var/www/images/;
}

Giai đoạn 2: Reverse Proxy

Học cách trỏ Nginx về một ứng dụng chạy Node.js hoặc Python đang listen ở port 3000 hoặc 8000.

Giai đoạn 3: Bảo mật & Tối ưu

Cấu hình SSL (HTTPS) với Let's Encrypt và bật Gzip compression để tăng tốc độ tải trang.

Hãy bắt đầu ngay từ hôm nay bằng việc tự build cho mình một chiếc Server nhỏ nhé!
